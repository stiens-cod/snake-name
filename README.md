# 贪吃蛇小游戏

这是一个基于 HTML、CSS 和 JavaScript 的贪吃蛇小游戏。玩家可以通过 `WASD` 键控制方向，游戏支持得分统计和重新开始功能。

## 特性

- `WASD` 控制方向
- 得分显示
- 最高分显示
- `重新开始` 按钮和空格键重新开始

## 运行方式

1. 打开 `index.html`
2. 在浏览器中使用 `WASD` 控制游戏

## 文件说明

- `index.html` - 游戏界面
- `style.css` - 页面样式
- `script.js` - 游戏逻辑
- `.gitignore` - Git 忽略配置
- `README.md` - 项目说明文档
