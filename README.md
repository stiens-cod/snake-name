# 贪吃蛇小游戏

这是一个基于 HTML、CSS 和 JavaScript 的贪吃蛇小游戏。玩家可以通过 `WASD` 键控制方向，游戏开始速度非常慢，并且每吃一个食物会逐渐提升速度。

## 新增功能

- 暂停 / 继续功能（按 `P`）
- 当前速度显示
- 游戏状态提示
- 更加现代的界面布局和视觉样式

## 特性

- `WASD` 控制方向
- 得分显示
- 最高分显示
- `重新开始` 按钮和空格键重新开始
- 起始速度很慢，随着每次吃食物逐渐变快

## 运行方式

1. 打开 `index.html`
2. 在浏览器中使用 `WASD` 控制游戏
